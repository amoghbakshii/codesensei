### <p align="center">Get It Running Locally</p>

<details>
<summary><strong>Click to view Backend Setup Instructions (Python & FastAPI)</strong></summary>

1.  **Navigate to the backend directory:**
    ```sh
    cd backend
    ```
2.  **Create and activate a virtual environment:**
    ```sh
    # For macOS/Linux
    python3 -m venv venv
    source venv/bin/activate

    # For Windows
    python -m venv venv
    .\venv\Scripts\activate
    ```
3.  **Install dependencies:**
    ```sh
    pip install -r requirements.txt
    ```
4.  **Set up environment variables:**
    Create a `.env` file in the `backend` directory and add your Google Gemini API key:
    ```
    GOOGLE_API_KEY="your_gemini_api_key_here"
    ```
5.  **Run the server:**
    ```sh
    uvicorn main:app --reload
    ```
    âœ… The backend is now live at `http://127.0.0.1:8000`.

</details>

<details>
<summary><strong>Click to view Frontend Setup Instructions (React & Vite)</strong></summary>

1.  **Navigate to the frontend directory:**
    ```sh
    cd frontend
    ```
2.  **Install dependencies:**
    ```sh
    npm install
    ```
3.  **Set up environment variables:**
    Create a `.env.local` file in the `frontend` directory to point to your local backend:
    ```
    VITE_API_BASE_URL="[http://127.0.0.1:8000](http://127.0.0.1:8000)"
    ```
4.  **Run the development server:**
    ```sh
    npm run dev
    ```
    âœ… The frontend is now live at `http://localhost:5173`. Open this URL in your browser.

</details>
